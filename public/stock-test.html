Base URL: http://127.0.0.1:8000/api

1. Get Stock List (GET)

GET Get stock list for a specific date
http://127.0.0.1:8000/api/stock-list?date=2025-11-21

Change the date parameter to test different dates

2. Daily Report (GET)

GET Get daily report with quantity changes
http://127.0.0.1:8000/api/stock-daily-report?date=2025-11-22

Shows quantity changes from previous day

3. Weekly Report (GET)

GET Get weekly report
http://127.0.0.1:8000/api/stock-weekly-report?week_start=2025-11-18

4. Monthly Report (GET)

GET Get monthly report
http://127.0.0.1:8000/api/stock-monthly-report?month=2025-11

5. Stock Summary (GET)

GET Get stock summary by date range
http://127.0.0.1:8000/api/stock-summary?from_date=2025-11-01&to_date=2025-11-30

6. POST Requests (Use Browser Console or Postman)

For POST requests, you need to use browser console or a tool like Postman. Here are examples:

Add Single Stock

POST: http://127.0.0.1:8000/api/stock-add Content-Type: application/json { "brand": "samsung", "size": "256gb", "color": "black", "quantity": 5, "stock_date": "2025-11-21" }

Bulk Add Stocks

POST: http://127.0.0.1:8000/api/stock-bulk-add Content-Type: application/json { "stock_date": "2025-11-21", "stocks": [ {"brand": "samsung", "size": "256gb", "color": "black", "quantity": 5}, {"brand": "apple", "size": "256gb", "color": "gold", "quantity": 3}, {"brand": "nokiya", "size": "256gb", "color": "blue", "quantity": 7} ] }

Test POST in Browser Console

// Copy and paste this in browser console (F12) fetch('http://127.0.0.1:8000/api/stock-bulk-add', { method: 'POST', headers: { 'Content-Type': 'application/json', }, body: JSON.stringify({ stock_date: '2025-11-21', stocks: [ {brand: 'samsung', size: '256gb', color: 'black', quantity: 5}, {brand: 'apple', size: '256gb', color: 'gold', quantity: 3}, {brand: 'nokiya', size: '256gb', color: 'blue', quantity: 7} ] }) }) .then(response => response.json()) .then(data => console.log(data)) .catch(error => console.error('Error:', error));
